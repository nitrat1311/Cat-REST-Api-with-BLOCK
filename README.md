# CAT APP #

A simple Flutter app showing cat images from CatApi


### USE ###

* Bloc
* REST API
* Native Splash Screen
* HTTP
* Connectivity
* [Cat Api](https://thecatapi.com/)
* shared_preferences
* json_serializer
